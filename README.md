# 👟 Colombia Shoes

Sitio web desarrollado como proyecto académico con el objetivo de simular una tienda virtual de calzado. El proyecto ofrece una interfaz intuitiva para que los usuarios puedan navegar por diferentes categorías de productos y conocer la información de la empresa.

## 📖 Descripción

Colombia Shoes es una aplicación web estática enfocada en presentar un catálogo de productos de forma organizada y atractiva. Durante su desarrollo se aplicaron conocimientos de estructura web, diseño responsivo y organización de archivos utilizando HTML, CSS y JavaScript.

## 🚀 Características

- Página principal con navegación intuitiva.
- Catálogo de productos.
- Sección de asesores.
- Organización por categorías.
- Diseño personalizado mediante CSS.
- Recursos multimedia (imágenes e íconos).
- Navegación entre diferentes páginas.

## 🛠️ Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript
- Google Fonts

## ▶️ Cómo ejecutar el proyecto

1. Clona el repositorio:

```bash
git clone https://github.com/kendarg/kendarg.git
```

2. Ingresa a la carpeta del proyecto.

3. Abre el archivo **index.html** en cualquier navegador web moderno.

También puedes utilizar la extensión **Live Server** de Visual Studio Code para una mejor experiencia durante el desarrollo.

## 🎯 Objetivos del proyecto

- Practicar la creación de interfaces con HTML y CSS.
